# MOC - Calidad de Código

Relaciones y conceptos compartidos entre herramientas de esta categoría.

## Conexiones

### Biome ↔ VS Code
- Biome tiene extensión oficial para VS Code
- Formateo y linting automático al guardar archivos
- Biome reemplaza ESLint + Prettier en proyectos JS/TS

### Biome ↔ ESLint / Prettier
- Biome unifica formateo y linting en una sola herramienta
- Migración directa desde ESLint + Prettier con `biome migrate`
- Compatible con proyectos existentes mediante configuración progresiva
