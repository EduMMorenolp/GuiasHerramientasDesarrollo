# Guías de Herramientas de Desarrollo

Bienvenido a las guías progresivas para dominar herramientas de desarrollo modernas. Cada herramienta está organizada en módulos que van desde fundamentos hasta temas avanzados.

## Categorías

### 01 - Editores y Terminal
Editores de código, terminales y productividad.

### 02 - Lenguajes y Runtimes
Node.js, Python, PM2, pnpm, Bun.

### 03 - Control de Versiones y CI/CD
Git, GitLab, GitHub Actions.

### 04 - Infraestructura
Docker, Nginx, Quartz.

### 05 - Bases de Datos
SQL, PostgreSQL, Redis.

### 06 - APIs y Testing
Postman, Playwright.

### 07 - IA
OpenCode, Ollama, Hermes Agent, OpenClaw.

### 08 - Calidad de Código
Biome.
