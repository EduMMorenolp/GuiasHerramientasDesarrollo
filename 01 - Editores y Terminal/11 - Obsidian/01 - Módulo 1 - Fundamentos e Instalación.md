# Módulo 1: Fundamentos e Instalación

**Objetivo**: Aprender y dominar: Fundamentos e Instalación.

## ¿Qué es Obsidian?
Obsidian es una aplicación de toma de notas basada en archivos Markdown locales. Permite crear una base de conocimiento personal interconectada mediante enlaces entre notas.

## Vaults
Un vault es una carpeta que contiene tus archivos Markdown. Obsidian trata esta carpeta como su espacio de trabajo. Puedes tener múltiples vaults y abrirlos independientemente.

## Markdown en Obsidian
Obsidian usa Markdown estándar con extensiones: encabezados, negritas, cursivas, listas, tablas, código, y enlaces internos [[wikilinks]].

## Editor: Live Preview, Source y Reading
- **Live Preview**: Muestra el formato visual mientras escribes, similar a un editor WYSIWYG.
- **Source**: Muestra el código Markdown sin procesar.
- **Reading**: Vista previa del documento renderizado, útil para lectura.

## Paleta de comandos (Ctrl+P)
La paleta de comandos permite acceder rápidamente a cualquier función de Obsidian: crear notas, cambiar ajustes, ejecutar plugins, etc. Se abre con Ctrl+P (Cmd+P en macOS).

## Instalación

### Windows
Descargar el instalador desde obsidian.md y ejecutarlo. También disponible via winget.

### macOS
Descargar el archivo DMG desde obsidian.md o usar Homebrew: `brew install --cask obsidian`.

### Linux
Disponible como AppImage, Flatpak (flathub) o paquete Snap.

### Móvil (Android/iOS)
Disponible en Google Play Store y Apple App Store. La sincronización entre dispositivos requiere Obsidian Sync o un servicio externo.
