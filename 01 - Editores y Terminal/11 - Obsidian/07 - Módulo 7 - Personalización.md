# Módulo 7: Personalización

**Objetivo**: Aprender y dominar: Personalización.

## Temas
Los temas modifican la apariencia completa de Obsidian. Se instalan desde Ajustes > Apariencia > Themes > Explorar. Existen cientos de temas creados por la comunidad.

## CSS Snippets
Fragmentos de CSS que sobreescriben estilos especificos sin modificar el tema activo.

### Como usar
1. Crear archivo `.css` en la carpeta `.obsidian/snippets/`.
2. Ir a Ajustes > Apariencia > CSS snippets.
3. Activar el snippet deseado.

### Ejemplos comunes
- Ocultar la cinta izquierda.
- Cambiar el color de los encabezados.
- Ajustar el espaciado entre lineas.

## Hotkeys
Todas las acciones de Obsidian se pueden asignar a combinaciones de teclas. Se configuran en Ajustes > Hotkeys.

### Acciones populares
| Accion                    | Hotkey predeterminado |
| ------------------------- | --------------------- |
| Paleta de comandos        | Ctrl+P                |
| Buscar archivos           | Ctrl+O                |
| Crear nota                | Ctrl+N                |
| Alternar vista            | Ctrl+E                |
| Backlinks                 | Ctrl+Shift+8          |

## Config JSON (.obsidian/)
La carpeta `.obsidian/` contiene la configuracion del vault en archivos JSON: `app.json`, `appearance.json`, `hotkeys.json`, `community-plugins.json`.

Se pueden editar manualmente para ajustes avanzados no expuestos en la interfaz.

## Workspace Layout
Los layouts de espacio de trabajo guardan la disposicion de paneles. Se guardan y cargan desde la paleta de comandos: "Manage workspace layouts".

## Multi-vault
Obsidian permite abrir multiples vaults simultaneamente en ventanas separadas. Los ajustes y plugins son independientes por vault.
