# Nginx — Guía completa

**Nginx** es un servidor web, proxy inverso y balanceador de carga de alto rendimiento. Se utiliza para servir contenido estático, enrutar tráfico a aplicaciones backend, terminar SSL/TLS, cachear respuestas y distribuir carga.
