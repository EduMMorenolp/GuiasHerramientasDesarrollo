# pnpm — Guía completa

**pnpm** es un gestor de paquetes rápido y eficiente para JavaScript/TypeScript. Usa un store global con enlaces simbólicos para ahorrar espacio, instala paquetes en paralelo y soporta monorepos con workspaces de forma nativa.
